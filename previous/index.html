GPS Fleet Tracking This GPS tracking web application for vehicle fleets allows users to effortlessly and in real-time locate their Fleet across North America. Industry defining software gives a comprehensive GPS tracking solution that is both powerful and easy to use. An easy-to-use web based user interface continuously and in real-time monitor fleets location, speed, stop times and activities. It also offers intuitive features including daily, weekly and monthly reporting and dispatching functions designed to increase efficiencies and reduce costs.

Power Equipment Monitoring It's twitter for panel boards. This cloud application receives or optionally polls the status, alarms and other data in the ModBus controllers of electrical equipment across multiple locations. This data is aggregated and presented as useful and actionable information in the form of visual charts and grids. The applications supports multiple types of equipment and customized register mapping and labeling. The application provides user interfaces for both the desktop and mobile devices.

Customized E-Commerce Platform This custom e-commerce platform provides state-of-the-art web features and performance while integrating with a legacy industry-specific ERP. A highly stylized multimedia user experience with a blazing fast shopping engine is combined with back office content management, warehouse fulfillment, payment processing and powerful reporting. The platform is also internationalized, servicing customers in multiple languages and currencies from multi-continent distribution centers. The solution also integrates with popular social media platforms and external sales channels. Totally baller.

Driver Performance Game Influencing driver habits with a unique gaming experience. This custom mobile came application tracks data reported by installed vehicle sensors and transforms it into concrete driver behavior scores via customizable scoring algorithms. Drivers are organized into teams that participate in seasons and rankings that provide an opportunity to offer tangible incentives to change bad habits. If you come in first I will give you this delicious doughnut.

CRM Workflow This system delivers awesome call-center efficiency and scalability. The automation, tele-prompting and process mapping facilitates proactive customer relationship management and campaigning. The built-in performance metrics give management instant feedback about productivity and efficacy. The system integrates with legacy ERP via SOAP and a VOIP auto-dialer. Killer app.

GPS Asset Tracking This GPS Asset Tracking web application a provides sophisticated enterprise resource tracking with advanced grouping, hierarchies, dynamic asset attributes and other powerful data management features. The application provides cutting-edge reporting, alerting and security features to organizations tracking thousands of trailers, trucks and other mobile assets. The solution includes both a rich user interface and a complete REST API.

Advanced Engineering Calculations This custom engineering application models polymer flow using a variety of sophisticated and user configurable algorithms. The app determines the significant factors necessary for a uniform dynamic die body deflection across the die width and reduces the time required to adjust the die to within target parameters with the cast film die. Yikes.
